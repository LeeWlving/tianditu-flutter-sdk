## 0.1.0

* 新增跨平台 `TiandituMap` 地图组件
* 支持矢量、影像、地形底图和中文注记
* 支持标注、折线、多边形、圆形覆盖物及地图控制器
* 新增一次定位、持续定位、权限和定位服务状态 API
* 新增地图用户位置蓝点和定位精度圈
* 新增圆形、多边形地理围栏及前台进入/离开事件
* 新增普通、区域、周边、视野和多边形 POI 搜索快捷 API
* 新增类型安全的驾车/步行路线规划参数和路线坐标解析
* 新增路线折线转换和地图范围自适应控制器
* 天地图瓦片请求支持 t0-t7 子域轮询
* 重构 Web Service 网络层，支持 HTTP Client 注入、超时和资源释放
* 所有请求参数改用安全 URI 编码
* 补齐公共 API 导出、地图示例、平台权限、能力矩阵和自动化测试
